# GameBanana 批量下载助手 (GameBanana Mod Downloader)

这是一个基于 Python 的自动化爬虫工具，专为 GameBanana (GB) 网站设计。它可以根据你输入的角色名字，自动搜索并批量下载相关的 Mod、预览图以及详细信息。

**核心特性：**

* **📂 自动整理**：自动创建以“角色名”为主目录、“Mod 标题”为子目录的文件夹结构。
* **🚀 进度条显示**：下载过程中显示实时进度条、下载速度和剩余时间，且下载完成后进度条**保留**在屏幕上，方便查看已完成列表。
* **💾 断点记忆**：内置“记忆功能”。如果任务中断（如断网或手动停止），下次运行时会自动跳过已下载完成的 Mod，绝不重复下载。
* **🖼️ 完整抓取**：不仅下载 Mod 本体文件，还会下载所有预览图片并生成 `info.txt` 信息文件。

## 1. 环境准备

在使用本工具前，请确保你的电脑上安装了 Python。

### 安装依赖库

本工具需要使用 `requests` (用于网络请求) 和 `tqdm` (用于显示进度条)。请在终端（CMD 或 PowerShell）中运行以下命令进行安装：

```bash
pip install requests tqdm

```

---

## 2. 如何使用

1. **保存代码**：将 Python 代码保存为文件，例如 `gb_downloader.py`。
2. **运行脚本**：在终端中运行该脚本：
```bash
python gb_downloader.py

```


3. **输入角色名**：
程序启动后会提示：
`请输入角色名字 (例如 Aino):`
在此处输入你想下载的角色英文名（支持模糊搜索），按回车确认。
4. **开始挂机**：
程序会自动翻页扫描，发现新的 Mod 会自动下载。你可以看到如下进度条：
```text
下载: MyModFile.zip...: 100%|██████████| 15.0M/15.0M [00:05<00:00, 2.80MB/s]

```

## 3. 文件保存结构

下载完成后，你的文件夹目录结构将如下所示：

```text
当前目录/
└── GameBanana_Downloads/          <-- 默认根目录
    └── Aino/                      <-- 你输入的角色名
        ├── history.txt            <-- [重要] 记录已完成的ID，请勿删除
        ├── Blue Archive Aino/     <-- 单个 Mod 的文件夹
        │   ├── info.txt           <-- Mod 描述、作者、点赞数等信息
        │   ├── preview_0.jpg      <-- 预览图 1
        │   ├── preview_1.jpg      <-- 预览图 2
        │   └── Aino_Mod.zip       <-- Mod 本体文件
        └── Another Mod Name/
            └── ...

```

## 4. 常见问题 (Q&A)

### Q1: 如果下载了一半我关掉了程序，下次怎么办？

**A:** 直接重新运行程序，输入相同的角色名字即可。
程序会读取 `history.txt`，瞬间跳过所有之前已经下载完成的 Mod，从你没下载的那个开始继续。

### Q2: 为什么有些 Mod 被跳过了？

**A:** 有两种情况：

1. **历史记录**：该 Mod 的 ID 存在于 `history.txt` 中，表示之前已经处理过了。
2. **文件已存在**：文件夹里已经有了同名文件，为了防止覆盖，程序会自动跳过。

### Q3: 我想下载其他游戏（不是原神）的 Mod 怎么办？

**A:** 打开代码文件，找到顶部的配置区域：

```python
GAME_ID = 8552  # 默认为原神

```

将 `8552` 修改为目标游戏的 ID 即可（例如 CS2 是 4248，可以在 GameBanana 对应游戏主页 URL 中找到）。

### Q4: 进度条太多刷屏了怎么办？

**A:** 这一版代码特意设置了进度条保留（`leave=True`），目的是让你清楚地看到哪些文件下载成功了。如果你不想要这个功能，可以联系开发者（AI）修改代码中的 `leave=True` 为 `leave=False`。

## 5. 注意事项

* **网络问题**：由于 GameBanana 服务器在海外，如果下载速度过慢或频繁报错，建议使用网络代理工具。
* **磁盘空间**：如果 Mod 数量很多（如 100+ 个），请确保硬盘有足够的剩余空间，图片和压缩包可能会占用数 GB 容量。
* **分类说明**：脚本默认搜索 `Mod` 分类。虽然现在 GameBanana 很多皮肤也归类在 Mod 下，但极少数被错误归类为其他类型的资源可能无法被搜到。
